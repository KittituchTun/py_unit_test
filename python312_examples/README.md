Python 3.12 Learning Examples
=================================

This project is a hands-on set of small, focused Python 3.12 examples.

How to run
----------

1) Ensure you have Python 3.12 available.
2) From the project root, run:

```
python3.12 main.py
```

You can also run a single example module directly, e.g.:

```
python3.12 01_basics.py
```

What you'll learn
-----------------

- Basics: variables, f-strings, pathlib
- Collections: list/dict/set comprehensions, Counter, default dict
- Functions: positional-only, keyword-only, pattern matching
- Classes & dataclasses: __repr__, ordering, immutability
- Structural pattern matching (match/case)
- Typing in Python 3.12, including PEP 695 syntax (type parameters)
- Asyncio with TaskGroup
- itertools.batched (new in 3.12)

File overview
-------------

- `main.py` — runner that executes all examples in order
- `01_basics.py` — language basics
- `02_collections.py` — core collections and tricks
- `03_functions.py` — function features
- `04_classes_dataclasses.py` — classes and dataclasses
- `05_pattern_matching.py` — match/case
- `06_typing_pep695.py` — typing with PEP 695
- `07_asyncio.py` — asyncio TaskGroup
- `08_itertools_batched.py` — itertools.batched
